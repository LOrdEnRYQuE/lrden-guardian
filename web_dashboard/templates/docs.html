Integrate LRDEnE Guardian into your applications

API Information

POST /analyze

Analyze text content for safety and hallucinations

Request Body

{
  "content": "Text to analyze...",
  "context": {
    "domain": "technical",
    "content_type": "documentation"
  }
}

Response

{
  "is_safe": true,
  "risk_level": "low",
  "confidence_score": 0.85,
  "guardian_score": 0.958,
  "analysis_summary": "Content appears safe...",
  "recommendations": ["Continue monitoring"],
  "detected_issues": [],
  "uncertainty_areas": [],
  "timestamp": "2026-02-21T13:00:00"
}

POST /analyze-file

Analyze uploaded file content

Note: Maximum file size is 16MB. Supported formats: .txt, .md, .csv

Request (multipart/form-data)

file: [File content]

GET /api-info

Get API and Guardian system information

Code Examples

Python

import requests

# Analyze content
response = requests.post('http://localhost:5000/analyze', json={
    'content': 'Your AI content here...',
    'context': {'domain': 'technical'}
})

result = response.json()
print(f"Safe: {result['is_safe']}")
print(f"Guardian Score: {result['guardian_score']}")

JavaScript

// Analyze content
const response = await fetch('http://localhost:5000/analyze', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
    },
    body: JSON.stringify({
        content: 'Your AI content here...',
        context: { domain: 'technical' }
    })
});

const result = await response.json();
console.log('Safe:', result.is_safe);
console.log('Guardian Score:', result.guardian_score);

cURL

curl -X POST http://localhost:5000/analyze \
  -H "Content-Type: application/json" \
  -d '{
    "content": "Your AI content here...",
    "context": {"domain": "technical"}
  }'
